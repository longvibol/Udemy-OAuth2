a Collection is essentially a specialized container—an object that groups multiple elements into a single unit.

Ex: បើយើងប្រើ Array យើងត្រូវបង្កើត Size វាមុន ចឹងបើវាត្រូវការបន្ថែមទៀត វានឹងពិបាកធ្វើការកែប្រែ តែបើយើងបង្កើតវាក្នង Container then put the element inside that container it can manage. 


1.1 The Collection Interface and the Collections Framework.


- The Interface: A single "blueprint" that defines the basic operations (adding, removing, clearing) for most data structures in Java.


- The Framework: A unified architecture that includes interfaces, implementations (classes like ArrayList), and algorithms (like sorting or searching).
