- Read Id from the jwt

- User Id :  "sub": "80373687-a5e1-43d6-bbd8-36d421e760c5" it is from JWT token 

- In order to inject (យកវាមកប្រើ)

==> We use authentication principle 


    @PreAuthorize("hasAuthority('ROLE_developer') or #id == #jwt.subject")
    @DeleteMapping(path="/{id}")

    public String deletedUser(@PathVariable String id, @AuthenticationPrincipal Jwt jwt){
        return "Deleted user with id: " + id + " and JWT subject " + jwt.getSubject();
    }

}
